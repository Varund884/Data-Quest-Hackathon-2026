# Trackfit

## Description
Trackfit is a soccer schedule optimizer designed for healthcare and wellbeing. Users can input their personal information and soccer schedule, and the application will optimize training days and rest periods based on individual stats, such as injuries, to minimize injury risk and ensure peak performance during matches.

## Features
- User input for personal stats and soccer schedules
- Optimization algorithm for training and rest days
- Focus on injury prevention and wellbeing
